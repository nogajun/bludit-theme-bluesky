Bluesky theme for bludit CMS
========================================

Blusky is [bludit CMS](https://www.bludit.com/) theme.

Features
----------------------------------------

* bootstrap 5.2.0
* Japanese support (include Japanese font)

Compatible
----------------------------------------

* Bludit v3.1.x (bludit v 4.0 is not supported.)
